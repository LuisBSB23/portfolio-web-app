# Portfólio Web - Luis Otavio Ferreira Dias

Bem-vindo ao repositório do meu portfólio web! Este projeto foi desenvolvido para apresentar meus projetos, certificações (como AWS Cloud Foundations e Scrum) e habilidades no desenvolvimento de software, destacando conhecimentos em tecnologias como Python, Node.js, Laravel e o ecossistema Next.js.

## 🚀 Como executar o projeto

A aplicação foi construída utilizando o framework Next.js. Siga o passo a passo abaixo para rodá-la localmente no seu VS Code.

### Pré-requisitos

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em seu computador.

### Passo a Passo

1. **Clone o repositório**
   Abra o terminal e faça o clone do projeto (caso ainda não tenha feito):
   ```bash
   git clone https://github.com/luisbsb23/portfolio-web-app.git
   ```

2. **Acesse a pasta do frontend**
   O código principal da aplicação está localizado dentro do diretório `frontend`:
   ```bash
   cd portfolio-web-app/frontend
   ```
   *(Nota: Se você já abriu a pasta `portfolio-web-app` diretamente no VS Code, basta digitar `cd frontend` no terminal integrado).*

3. **Instale as dependências**
   Instale todos os pacotes necessários para o funcionamento do projeto:
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento**
   Rode o comando abaixo para iniciar a aplicação:
   ```bash
   npm run dev
   ```

5. **Acesse no navegador**
   Abra o seu navegador e acesse [http://localhost:3000](http://localhost:3000). A página será recarregada automaticamente sempre que você salvar alguma modificação nos arquivos do projeto.
